Tekstbestanden lezen met File.ReadAllText() in C#

Wat is File.ReadAllText()?

File.ReadAllText() is een methode in C# waarmee je heel eenvoudig de volledige inhoud van een tekstbestand in Ã©Ã©n keer kunt inlezen. Dit betekent dat je niet handmatig een bestand hoeft te openen, regels hoeft te lezen of te sluiten; alles wordt automatisch voor je geregeld.

Deze methode leest het bestand als Ã©Ã©n grote tekststring. Dat is ideaal als je bijvoorbeeld snel een configuratiebestand of een kleine tekst wilt verwerken zonder veel ingewikkelde code.

Het is belangrijk dat het bestand bestaat, anders krijg je een foutmelding. Daarom kun je het beste altijd controleren of het bestand bestaat, of gebruik maken van try-catch om fouten netjes af te handelen.

Een eenvoudig voorbeeld

Hier is een voorbeeld hoe je File.ReadAllText() gebruikt om een tekstbestand te lezen en de inhoud weer te geven in de console:

using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Het pad naar het bestand dat je wilt lezen
        string pad = "voorbeeld.txt";

        try
        {
            // Lezen van de volledige inhoud als string
            string inhoud = File.ReadAllText(pad);

            // Weergeven van de inhoud in de console
            Console.WriteLine("Inhoud van het bestand:");
            Console.WriteLine(inhoud);
        }
        catch (FileNotFoundException)
        {
            Console.WriteLine($"Het bestand '{pad}' is niet gevonden.");
        }
        catch (Exception ex)
        {
            Console.WriteLine("Er is een fout opgetreden bij het lezen van het bestand:");
            Console.WriteLine(ex.Message);
        }
    }
}

In dit voorbeeld wordt het bestand voorbeeld.txt gelezen en wordt de inhoud netjes op het scherm getoond. Als het bestand niet bestaat, krijg je een vriendelijke foutmelding.

Waar staat het bestand?

Net als bij schrijven staat het bestand meestal in de werkmap van je applicatie, bijvoorbeeld bin\Debug\netX.X bij Visual Studio projecten. Als je zeker wilt weten waar het programma zoekt, kun je het volgende gebruiken: 

Console.WriteLine("Huidige werkmap: " + Directory.GetCurrentDirectory());

Dit helpt je om het bestand snel te vinden, zodat je het kunt openen of aanpassen. Als het bestand er niet is, kun je het handmatig toevoegen of eerst een schrijfoperatie uitvoeren om het aan te maken.
