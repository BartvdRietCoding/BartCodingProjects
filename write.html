Tekstbestanden schrijven met File.WriteAllText() in C#

Wat is File.WriteAllText()?

Een eenvoudig voorbeeld

Hier is een korte codevoorbeeld dat aantoont hoe je File.WriteAllText() gebruikt om een bestand te schrijven:

using System;
using System.IO;

class Program
{
    static void Main()
    {
        // De naam van het bestand dat we willen schrijven
        string pad = "voorbeeld.txt";

        // De tekst die we in het bestand willen plaatsen
        string tekst = "Hallo wereld! Dit is een voorbeeldtekst.";

        // Schrijven van de tekst naar het bestand
        File.WriteAllText(pad, tekst);

        Console.WriteLine($"De tekst is succesvol weggeschreven naar {pad}");
    }
}

Wanneer je dit programma uitvoert, wordt het bestand voorbeeld.txt aangemaakt in dezelfde map waar je programma draait.

Waar vind ik het bestand terug?

De locatie waar het bestand wordt opgeslagen is belangrijk om te weten, zeker als je het resultaat wilt bekijken. Standaard schrijft File.WriteAllText() het bestand in de huidige werkmap van je applicatie.

Als je werkt in Visual Studio, is dit vaak de bin\Debug\netX.X map van je project. Je kunt dit pad gemakkelijk achterhalen met de code: 

Console.WriteLine(Directory.GetCurrentDirectory());

Het programma geeft dan het volledige pad naar de map waar het bestand wordt opgeslagen. Zo weet je precies waar je het bestand kunt openen, bijvoorbeeld via de Verkenner in Windows of Finder op macOS.
